# Function Docs

Per-function reference pages for the UniSparse toolbox.

## How To Read These Docs

Each page follows this structure:
- What this function does
- Signature
- Typical use case
- Mathematical form
- Parameters
- Returns
- Example
- Practical notes

## Recommended Reading Order

1. Start with unisparse for full pipeline usage.
2. Read objective pages for penalty-specific behavior.
3. Use utility and metrics pages while evaluating experiments.
